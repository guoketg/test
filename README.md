# Modern-CNN-Architectures

PyTorch Implementations and Performance Comparisons of Classic and Modern CNN Architectures

## Project Overview
This project contains implementations of various mainstream deep learning vision models, aiming to demonstrate the design concepts and performance differences of different network architectures. All models use a unified data loading and training framework for convenient comparative experiments.

Supported Models:
- AlexNet
- VGG-11
- ResNet
- DenseNet
- GoogleNet (Inception v1)
- EfficientNet (B0/B1)
- Vision Transformer (ViT)

Supported Datasets:
- CIFAR10
- FashionMNIST
- MNIST

## Environment Setup
```bash
pip install -r requirements.txt
```
